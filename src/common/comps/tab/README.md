# wxc-tab

> 小程序组件 -


```



| 名称                  | 描述                         |
|----------------------|------------------------------|
|`defaultIndex`        | 默认显示第几个tab [Number]  ( 0 )
|`animate`             | 底线是否应用动画 [Boolean] (true)
|`tabs`                | tab列表 [Array] {key:1,label:'Tab一'}] key字段不能变
|`textColor`           | 字体颜色 [String]（#666）
|`activeTextColor`     | 字体高亮颜色 [String]
|`activeLineColor`     | 底线高亮颜色 [String]


bind-methods : onSwitch  => key 当前选中的key值
